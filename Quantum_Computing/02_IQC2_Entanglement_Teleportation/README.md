# IQC-2 - Entanglement and Teleportation

Python/Qiskit Aer implementation of Bell-state measurements, chi-square tests, superdense coding, dynamic-circuit teleportation and binomial tests.

Run:

```bash
python iqc2_entanglement_teleportation.py
```

The simulation uses 1000 shots and a fixed seed. Outputs are stored in `IQC2_results.json` and `figures/`.
