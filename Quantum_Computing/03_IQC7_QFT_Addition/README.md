# IQC-7 - QFT Addition

Python/Qiskit implementation of teleportation verification and an eight-qubit Draper QFT adder for
`(|2>|9> + |6>|5>)/sqrt(2)`.

Run:

```bash
python iqc7_qft_addition.py
```

The ideal results use Qiskit Aer. The second data set uses a local pedagogical noise model and is not an IBM hardware execution. Real hardware requires the student's IBM Quantum credentials and job ID.
