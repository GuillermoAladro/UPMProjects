# IQC-1 - Quantum Fundamentals

Python/Qiskit implementation of one-qubit gates, controlled operations, Bell states and the no-cloning theorem.

Run:

```bash
python iqc1_quantum_fundamentals.py
```

Outputs:
- `IQC1_results.json`
- figures in `figures/`
- `IQC1_Report.pdf`
- optional Jupyter notebook `iqc1_quantum_fundamentals.ipynb`
